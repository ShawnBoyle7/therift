## Technologies

### Front-End:
* Framework: React/Next.js
* Language: TypeScript
* Map Library: Konva.js
* Styling: Material UI
* Hosting: Vercel

## Back-End:
* Framework: Flask
* Database: PostgreSQL
* ORM: SQLAlchemy
* Authentication: Flask-Security
* Hosting: Heroku
